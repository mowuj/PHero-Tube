VAR:

  • Var variable is Functional scope.
  • Var variable can be updated and re-declared into the scope.
  • Var variables can be declared without initialization.

LET:

  • Variables declared with 'let' have block-level scope.
  • They are not accessible before the declaration.
  • let variable can be updated but cannot be re-declared into scope.
  • Let variables also can be declared without initialization.

CONST:

  • Variables declared with 'const' also have block-level scope
  • const variable cannot be updated or re-declared into scope.
  • const variables cannot be declared without initialization.

NULL:

  • Null is object type.
  • Null means intentional absence of the value.
  • When used in arithmetic operations,null will be converted to 0 behind the screens.

UNDEFINED:

  • Undefined is a type.
  • Undefined means, the value does not exist in the compiler.
  • When used in arithmetic operations, undefined will result in NaN.
